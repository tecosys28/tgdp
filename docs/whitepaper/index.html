Executive Summary

The Tokenized Gold Depository Protocol (TGDP) is a comprehensive ecosystem designed to unlock the latent value of household gold in India. By tokenizing physical gold into digital TGDPs (Tokenized Gold Depository Points), the platform enables zero-fee trading, FTR swaps, and participation in a licensed network spanning 164 nations.

Key Components

  • ✓ T-Gold: Gold tokenization and trading platform
  • ✓ GIC Licensing: Licensee network with 25% revenue share
  • ✓ T-JR: Jewelry return with LBMA-benchmarked valuation
  • ✓ T-JDB: IPR-protected design marketplace
  • ✓ Ombudsman: Independent dispute resolution
