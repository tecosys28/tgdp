Participants

Who Can File Complaints?

🏠
Households

File complaints against Licensees, Jewelers, Designers, or platform issues related to minting, trading, or FTR swaps.

📜
Licensees

File complaints regarding GIC disputes, household issues, payment discrepancies, or platform technical problems.

💎
Jewelers

File complaints about T-JR disputes, payment issues, customer conflicts, or platform operational matters.

🎨
Designers

File complaints regarding IPR violations, order disputes, payment issues, or T-JDB platform concerns.

💼
Consultants

File complaints about referral disputes, commission issues, or conflicts with other ecosystem participants.

📢
Advertisers

File complaints regarding advertising disputes, billing issues, or content-related conflicts.

Need to File a Complaint?

Access your portal dashboard to file a complaint. You can also track existing complaints and view resolution status.
